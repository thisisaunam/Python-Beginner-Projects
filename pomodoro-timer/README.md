# pomodoro timer

a command-line python program that simulates a basic pomodoro timer with theme selection and user input validation.

---

## features

version 1:

* allows user to select different study themes
* provides predefined timer options (25/5, 30/5, 50/10)
* includes a custom timer option (placeholder)
* validates user input using loops and try-except
* prevents invalid and non-integer inputs
* uses functions for modular program structure
* provides a clean and interactive CLI experience with formatted prompts

---

## how to run

run the file using:

```id="run101"
python pomodoro_timer_v1.py
```
